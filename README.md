# Acoustic-classification
A species classification example with acoustic characters using machine learning methods was presented. This approch can be apply to other creatures or soundscape analysis.

## Materials and Methods
### Data acquestion
Chirps of six species of crickets were selected owing to they are common and easy to find and collect. Sound recording was conducted using smartphone (Samsung Galaxy A52), and Audacity (https://audacityteam.org/), a free sound analysis software was used to check if files were availible.
